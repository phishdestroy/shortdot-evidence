# Legal Notice

## Nature of This Repository

This repository contains factual documentation of domain registrations, network data, HTTP response characteristics, and publicly observable abuse indicators in domain zones operated by ShortDot SA.

All data was collected via:
- NetAPI (WHOIS zone data — publicly licensed)
- Public DNS resolution
- HTTP GET requests to publicly accessible URLs
- Cross-reference with public threat intelligence databases (Spamhaus DBL, SURBL, URLhaus, ThreatFox)

No unauthorized access occurred. No credentials were used that were not publicly provided. No private systems were accessed.

## Accuracy

All factual claims in this repository are based on observed data. Where analysis involves estimates (wholesale pricing, revenue calculations), this is explicitly stated. Estimates are clearly labeled as estimates.

Domain classifications reflect observable technical indicators at the time of measurement. Classifications may change as domains change their content or status.

## Purpose

This repository is published for:
- Public interest research into domain zone abuse
- Threat intelligence sharing
- Evidence support for ICANN compliance processes
- Evidence support for law enforcement referrals
- Brand protection (UDRP / URS proceedings)
- Academic research
