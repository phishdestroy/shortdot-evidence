# Evidence Directory

Screenshots of confirmed phishing, drain, carding, and fraud pages hosted on ShortDot zones.

All screenshots are captured with headless Chromium (Playwright), saved as PNG, and SHA-256 hashed post-capture.

## Verify Integrity

```bash
sha256sum -c HASHES.txt
```

## Naming Convention

```
YYYY-MM-DD_domain_category.png
```

Example: `2026-06-15_chase.bond_PHISHING_FINANCE.png`
