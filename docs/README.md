# docs/

GitHub Pages source for the ShortDot SA investigation dashboard.

Served at: https://phishdestroy.github.io/shortdot-evidence/

| File | Purpose |
|---|---|
| `index.html` | Main investigation dashboard |
| `domains.html` | Searchable/filterable domain table |
| `build_datajson.py` | Converts data/index.json + ioc/indicators.csv → data.json |
| `data.json` | Pre-built data for the Pages frontend (auto-updated) |
| `robots.txt` | Search engine crawl policy |
| `sitemap.xml` | Sitemap for SEO |
| `llms.txt` | Machine-readable summary for LLM crawlers |
| `site.webmanifest` | PWA manifest |
| `.nojekyll` | Disables Jekyll processing on GitHub Pages |
